# Callouts v1.2.0

## Live events improvements

### Per-category log budgets
Each event category now keeps its own independent ring buffer, so a noisy category
(chat spam, buff-on-everyone casts) can never push out the events you actually care
about (enemy mechanic casts, debuffs on you or your party).

- Default limit: **20,000 entries per category** (up from a single shared 2,000).
- Per-category overrides in Settings → Limits → "Per-category limits": set each
  category to anything from 100 to 100,000, independently.
- "Reset all to default" button to clear all overrides at once.
- Filter labels in the Live events window now show live counts per category.

### Category filters & search bar
- **Category filters**: toggle enemy casts, other casts, self debuffs/buffs, party
  debuffs/buffs, other status, chat, duty, VFX, and head markers individually.
  All / None shortcuts included.
- **Search bar**: filter the visible log by text (case-insensitive).
- Footer shows matched / total counts and flags when the 1,000-row display cap
  would hide results (narrows with search).

## Upgrade notes
The old single `EventBufferSize` config key is silently ignored on upgrade; all
categories start at the new default (20,000). No data loss; no migration required.
