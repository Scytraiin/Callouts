First stable release: user-defined trigger rules turning in-game events into local alerts.

## Features
- **Trigger sources** — chat (contains/regex with captures), enemy/actor casts, status
  gained/removed (self / party / target), duty wipe & recommence, and an opt-in advanced
  tier (VFX spawns, head markers via a named-marker picker).
- **Outputs, per rule** — Echo (with placeholders), one of 16 sound effects, and toasts
  (Normal/Quest/Error), in any combination.
- **Authoring** — Rules window with a live tester, a Live events feed with one-click
  "create rule from this event", per-rule cooldowns, a global rate limit, zone/combat/duty
  scoping, bulk operations with undo, and test-fire.
- **Sharing** — import/export rule packs (clipboard, duplicate-safe) and a one-click
  starter pack.
- **Safety** — the plugin's own Echo output can never re-trigger a rule; a bad regex
  disables only its own rule; your config is backed up before any migration and downgrades
  are refused.

## Known limitations
- The stable cast source only sees actions with a cast bar (instant abilities are invisible).
- `/echo` macros are unmatchable (the Echo channel is excluded to prevent feedback loops).
- Advanced-tier sources rely on a memory signature that a game patch can break; when that
  happens they disable themselves and you get a one-time login notice.
