# Callouts v0.1.0-alpha

Walking-skeleton release: proves the delivery pipeline end to end. Not intended
for everyday use yet.

- Loadable Dalamud plugin (API level 15); `/callouts` toggles a stub window.
- Test-gated Docker build, CI (unit tests + Docker smoke build), and release
  tooling ported from the sibling plugins.
- No trigger rules yet — the first chat→Echo rule lands in the next release
  (see the roadmap in README.md).
